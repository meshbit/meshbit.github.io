## 🛠️ 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| **前端** | HTML5 + CSS3 + JavaScript (ES6+) | 单文件 SPA，无框架依赖 |
| **UI 框架** | Tailwind CSS (CDN) | 搜索页卡片布局 |
| **代理层** | Go 1.26 | 高性能反向代理 + 缓存 |
| **搜索引擎** | PanSou (Go) | TG 频道聚合搜索 |
| **容器化** | Docker | PanSou 运行环境 |
| **内网穿透** | Cloudflare Tunnel | 公网 HTTPS 访问 |
| **辅助脚本** | Python 3 | 爬虫框架、链接检测 |

## 🏗️ 架构设计

```
┌─────────────────────────────────────────────────────────┐
│                      用户浏览器                          │
└─────────────────────┬───────────────────────────────────┘
                      │ HTTPS
                      ▼
┌─────────────────────────────────────────────────────────┐
│              Cloudflare Tunnel (pan.okva.cc)             │
└─────────────────────┬───────────────────────────────────┘
                      │
                      ▼
┌─────────────────────────────────────────────────────────┐
│               Go 代理 (:8080)                            │
│  ┌─────────────┐  ┌──────────────┐  ┌────────────────┐  │
│  │ 静态文件服务  │  │ 10分钟LRU缓存 │  │ 失效链接过滤    │  │
│  └─────────────┘  └──────────────┘  └────────────────┘  │
│  ┌─────────────────────────────────────────────────────┐ │
│  │              后台管理系统 (/admin/)                   │ │
│  │  仪表盘 | 频道 | 插件 | 广告 | 导航 | 投稿 | 配置 | 缓存 │ │
│  └─────────────────────────────────────────────────────┘ │
└─────────────────────┬───────────────────────────────────┘
                      │
                      ▼
┌─────────────────────────────────────────────────────────┐
│            PanSou Docker (:8081)                         │
│  ┌──────────────────┐  ┌─────────────────────────────┐  │
│  │  174 个 TG 频道    │  │  88 个爬虫插件                │  │
│  │  (Telegram API)   │  │  (影视站/论坛/微博/磁力等)     │  │
│  └──────────────────┘  └─────────────────────────────┘  │
└─────────────────────────────────────────────────────────┘
```

### 数据流

```
用户搜索 "Python"
  → Go 代理检查缓存 (命中则直接返回)
  → 未命中: 转发到 PanSou API
  → PanSou 并发查询 174 个 TG 频道 + 88 个插件
  → Go 代理过滤失效链接
  → 写入10分钟缓存
  → 返回 JSON 给前端渲染
```

### 缓存策略

- **TTL:** 10 分钟
- **Key:** 搜索关键词
- **命中率:** 重复搜索秒级响应
- **失效链接过滤:** 从 `link_status.json` 加载已确认失效的 URL，搜索结果中自动剔除

## 🚀 本地部署

### 环境要求

- Windows 10/11 (x64)
- Docker Desktop
- Go 1.22+
- Cloudflare Tunnel (可选，用于公网访问)

### 1. 部署 PanSou 搜索引擎

```bash
# 拉取镜像
docker pull ghcr.io/fish2018/pansou-web

# 启动容器
docker run -d --name pansou \
  -p 8081:80 \
  --env-file pansou.env \
  --restart unless-stopped \
  -v D:\www\pansou\index.html://app//frontend//dist//index.html \
  ghcr.io/fish2018/pansou-web
```

### 2. 编译并启动 Go 代理

```bash
cd proxy
go build -o proxy.exe main.go

# Windows 启动
set PORT=8080
set STATIC_DIR=D:\www\pansou
proxy.exe
```

### 3. 配置 Cloudflare Tunnel

```yaml
# C:\Users\用户名\.cloudflared\config.yml
tunnel: <tunnel-id>
credentials-file: C:\Users\用户名\.cloudflared\<tunnel-id>.json

ingress:
  - hostname: pan.okva.cc
    service: http://localhost:8080
  - service: http_status:404
```

### 4. 开机自启

将 `proxy.exe` 快捷方式放入 Windows 启动文件夹:
```
C:\Users\<用户名>\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup\
```

## 🔧 维护指南

### 添加新频道
1. 访问后台 `/admin/` → "频道管理"
2. 在文本框中输入频道名（每行一个或逗号分隔）
3. 点击"添加"
4. 自动写入 `pansou.env` 并生效

### 修改首页内容
1. 后台 → "⚙ 配置"
2. 修改标语、Logo 文字、底部声明
3. 保存后首页自动更新

### 修改密码
1. 后台 → "⚙ 配置" → "修改密码"
2. 输入旧密码 + 新密码 → 保存

### 监控服务
- 代理每 5 分钟自动检测存活，挂了自动拉起
- 链接失效检测每天凌晨 3 点自动运行
